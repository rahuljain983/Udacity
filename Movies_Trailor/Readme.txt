Please follow the following steps to run the application:

1. Make sure you have python install on your machine.
2. Open the Movie Trailer Folder.
3. Double Click on entertainment_center.py to run the application.
4. A web page will open which will having the Poster image of Movies.
5. click on poster image to run the trailer of the movie.
6. A fresh_tomatoes.html file will be generated in the same folder after you run the movie Trailer website.
